# 增长体系设计能力

## 一、适用场景
- 从零搭建增长体系
- 设计分销/代理网络
- 用户激励机制（积分/等级/权益）
- 活动运营系统（拉新/促活/召回）
- 用户生命周期管理

## 二、核心模型

### 2.1 增长飞轮
```
更多用户 → 更多交易 → 更多数据
    ↑                    │
    │                    ▼
    └── 更好体验 ← 更好产品
```

### 2.2 分销网络设计
```
  ┌─────────┐
  │ 平台方   │ ← 定义规则、结算佣金
  └────┬────┘
       │
  ┌────▼────┐
  │ 一级代理 │ ← 直推佣金20%
  └────┬────┘
       │
  ┌────▼────┐
  │ 二级代理 │ ← 间推佣金5%
  └─────────┘
```

### 2.3 积分/等级体系
```
积分: 交易/签到/任务 → 累计积分 → 兑换权益/抵扣手续费
等级: 累计交易额 → 升级 → 解锁专属权益/费率优惠
```

## 三、设计步骤（SOP）
1. 确定增长北极星（DAU/GMV/交易笔数？）
2. 设计获客渠道矩阵
3. 设计激活漏斗（从注册到首次交易）
4. 设计留存机制（签到+推送+权益+教育）
5. 设计分销/裂变机制
6. 设计结算与防作弊

## 五、可复用案例
- 交易所邀请返佣：20%手续费返佣+下级5%
- 签到系统：连续7天签到奖励交易金
- 等级体系：VIP1-VIP10，按月度交易量升级
