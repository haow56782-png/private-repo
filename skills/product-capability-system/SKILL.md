# Product Capability System

可调用的产品能力系统。每个模块采用标准 5 层结构：

1. **适用场景** — 什么时候用这个模块
2. **核心模型** — 抽象公式和结构
3. **设计步骤（SOP）** — 可执行的步骤序列
4. **标准模板** — 字段结构 / 流程 / 表格
5. **可复用案例** — 来自真实项目的参考

- `user-role-modeling` — 用户与角色建模

- `risk-control-exception` — 风控与异常处理

- `product-review-framework` — 产品评审框架
